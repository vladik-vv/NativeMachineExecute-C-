**Native Execute Machine Assembly** — ассемблер для виртуальной машины из игры zabvensk.
Написан на языке C# с включенным PublishAOT

Запись всей разработки - https://www.youtube.com/playlist?list=PL7lwZv4fhGCbd4QIZXwvMrdQQru8_7TJH

## Версия 1.3

Стабильный релиз.

### Возможности:
- 8 регистров (r1-r5, rnd, rnr, rvc)
- Типы: byte, short, float, double, string, vec2/vec3/vec4
- Арифметика, тригонометрия, массивы, векторы
- Системные инструкции, ввод-вывод, память
- Одномерные массивы arr
- Двухмерные массивы mx2

### Документация html в релизе рядом с исполняемым файлом

В папке ASM находятся консольные программы написанные на nem
В папке Csharp находится открытый код самого интерпретатора
